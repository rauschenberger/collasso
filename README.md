# Sparse linear multi-task regression

This Python library implements "Sparse linear multi-task regression with correlation-based information sharing". This personal GitHub repository is mirrored at two institutional GitLab instances (see [LIH](https://git.lih.lu/arauschenberger/collasso) and [LCSB](https://gitlab.com/uniluxembourg/Personalfolders/armin.rauschenberger/collasso)).
